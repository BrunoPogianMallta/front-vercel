Teste Técnico Guimepa – Sistema de Blog com Autenticação

  • Descrição: Aplicação desenvolvida para demonstrar habilidades com PHP puro, organização de código e uso de Docker.
  • Sistema de autenticação completo (cadastro e login de usuários)
  • Funcionalidade de CRUD completo para postagens (criar, listar, editar e excluir)
  • Visualização de postagens em lista e detalhes individuais
  • Tecnologias utilizadas: PHP 8+, MySQL, Docker, Composer,JS
  • Extras: Estrutura HTML/CSS, PHPUnit para testes, migrations e documentação da API
